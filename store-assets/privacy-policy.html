개인정보처리방침

최종 업데이트: 2025년 5월

Bemarkable(이하 "개발자")은 오늘 뭐 입지?(이하 "앱")와 관련하여 이용자의 개인정보 보호를 중요하게 생각하며, 아래와 같이 개인정보처리방침을 안내합니다.

1. 수집하는 개인정보

개발자는 회원가입 등을 통해 이용자로부터 직접 개인정보를 수집하지 않습니다.

단, 서비스 제공을 위해 다음 정보가 자동 수집될 수 있습니다.

  • 위치 정보 — 현재 위치 기반 날씨 조회를 위해 GPS 위치를 사용합니다. 위치 정보는 날씨 API 조회 목적으로만 사용되며 서버에 저장되지 않습니다.
  • 광고 ID (Google AdMob) — 광고 서비스 제공을 위해 Android 광고 식별자가 자동 수집될 수 있습니다.
  • 기기 정보 — OS 버전, 기기 모델 등 (AdMob 자동 수집)
  • IP 주소 및 대략적 위치 — 광고 최적화 목적 (AdMob 자동 수집)

2. 수집 목적

  • 현재 위치 기반 실시간 날씨 조회 (기상청 단기예보 API)
  • 맞춤형 광고 및 일반 광고 제공 (Google AdMob)
  • 광고 성과 측정 및 사기 방지

3. 위치 정보 처리

앱은 현재 위치의 날씨를 불러오기 위해 기기의 GPS 위치 정보를 사용합니다.

  • 위치 정보는 기상청 날씨 API 호출 시에만 사용됩니다.
  • 개발자 서버에 위치 정보가 저장되거나 전송되지 않습니다.
  • 위치 권한을 허용하지 않아도 앱의 기본 기능을 이용할 수 있습니다 (지역 수동 선택 가능).

4. 제3자 서비스

앱은 다음 제3자 서비스를 사용합니다.

  • 기상청 단기예보 API — 날씨 데이터 제공 (공공데이터포털)
  • Google AdMob — 광고 서비스. Google LLC의 개인정보처리방침을 따릅니다.

관련 정책:

  • Google 개인정보처리방침
  • AdMob 파트너 정책

5. 데이터 보관 및 삭제

개발자가 직접 수집·보관하는 개인정보는 없습니다. AdMob을 통해 수집된 데이터는 Google의 정책에 따라 관리됩니다. 광고 개인화를 원하지 않는 경우 기기의 설정 → 광고 → 광고 개인화 선택 해제에서 설정할 수 있습니다.

6. 아동 개인정보 보호

본 앱은 만 13세 미만 아동을 대상으로 하지 않으며, 만 13세 미만 이용자로부터 의도적으로 개인정보를 수집하지 않습니다.

7. 개인정보처리방침 변경

본 방침은 법령 또는 서비스 변경 사항을 반영하기 위해 수정될 수 있습니다. 변경 시 이 페이지의 업데이트 날짜를 통해 고지합니다.
