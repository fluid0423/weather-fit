# 오늘 뭐 입지? — Play Store 등록 자료

## 앱 이름
오늘 뭐 입지?

## 짧은 설명 (80자 이내)
날씨 보고 옷 고르는 시간 끝! 기온에 딱 맞는 코디를 바로 확인하세요.

## 전체 설명 (4000자 이내)
오늘 아침 옷장 앞에서 뭘 입을지 고민한 적 있으신가요?

**오늘 뭐 입지?** 는 현재 날씨와 기온을 분석해서 딱 맞는 코디를 추천해주는 날씨 패션 앱이에요.

**✨ 주요 기능**

🌤️ 실시간 날씨 기반 코디 추천
기상청 단기예보 데이터를 기반으로 현재 기온·날씨에 맞는 옷차림을 즉시 추천해요.

👗 8단계 기온별 코디 가이드
- 28°C 이상 → 민소매·반바지
- 23~27°C → 반팔·얇은 하의
- 20~22°C → 얇은 긴팔·청바지
- 17~19°C → 긴팔·면바지
- 12~16°C → 가디건·니트
- 9~11°C → 재킷·청바지
- 5~8°C → 코트·울 소재
- 4°C 이하 → 패딩·두꺼운 레이어링

📍 위치 자동감지 + 지역 수동 선택
GPS로 현재 위치를 자동으로 감지하고, 원하는 지역을 직접 선택할 수도 있어요.
전국 17개 시/도, 모든 구/군/시 지원.

📅 이번 주 날씨 미리보기
일주일치 날씨와 코디를 미리 확인해서 여행·약속 준비를 스마트하게!

🎨 날씨별 동적 테마
맑은 날엔 따뜻한 오렌지, 비 오는 날엔 딥 블루, 흐린 날엔 차분한 그레이로 배경이 바뀌어요.

**📊 데이터 출처**
날씨 데이터는 기상청 단기예보 공공 API를 사용합니다.
